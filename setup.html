Launch your personal Stockkar Algo app.

Create one user-owned AWS app with its own static IP, frontend, backend, and protected one-click updates. The user approves AWS, but does not edit code.

Step 01

Deploy Personal App

Download the AWS template, then upload it in CloudFormation. AWS creates the frontend, backend, server, static IP, and updater together.

Download AWS Template Open AWS Create Stack
Choose Upload a template file, select the downloaded YAML, choose a private Update PIN, and keep the other defaults.
Step 02

Open AppUrl

Wait for CREATE_COMPLETE, open Outputs, and copy AppUrl. This is the user's permanent personal Stockkar link.

The same AppUrl serves frontend and backend. Updates can later be installed from Settings with the user's private Update PIN.
A separate Vercel deployment is no longer required for the user's trading app.
Step 03

Broker IP

After AWS setup, copy the static IP into Dhan/Zerodha/Upstox and save broker tokens inside the app.

Paste StaticIp from AWS Outputs. The personal App URL and Backend URL are generated automatically.
