Launch your personal Stockkar Algo app.

Create the user-owned frontend with Vercel, connect it to the user-owned backend, and finish with the broker static IP setup. The user approves each platform, but does not edit code.

Step 01

Deploy Frontend

Opens Vercel with Stockkar prefilled. User logs in, approves, and clicks Deploy.

Deploy to Vercel
Step 02

Deploy Backend

Download the backend template, then open AWS CloudFormation and upload it. AWS will create the server, static IP, and network automatically.

Download AWS Template Open AWS Create Stack
In AWS, choose Upload a template file, select the downloaded YAML, then continue. After stack is complete, copy StaticIp from Outputs.
If AWS asks for parameters, keep the defaults. No VPC or subnet selection is needed.
Step 03

Broker IP

After AWS setup, copy the static IP into Dhan/Zerodha/Upstox and save broker tokens inside the app.

Paste StaticIp from AWS Outputs. Backend URL is generated automatically, then opening the app auto-fills Settings.
